biosimulators_utils.kisao.utils.normalize_kisao_id(id)[source]¶

Normalize an id for a KiSAO term to the official pattern KISAO_\d{7}.

The official id pattern for KiSAO terms is KISAO_\d{7}. This is often confused with KISAO:\d{7} and \d{7}. This function automatically converts these other patterns to the offfical pattern.

Parameters

id (str) – offical KiSAO id with pattern KISAO_\d{7} or a variant such as KISAO:\d{7} or \d{7}

Returns

normalized KiSAO id that follows the official pattern KISAO_\d{7}

Return type

str
